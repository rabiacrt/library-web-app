
# Kütüphane

🔗 **Live Demo:** https://library-web-app-wine.vercel.app/

Bu proje, Project Gutenberg kütüphanesindeki binlerce ücretsiz kitaba erişim sağlayan, modern kullanıcı arayüzü ve akıcı deneyimi odak noktasına alan bir web uygulamasıdır. Kullanıcıların kitapları aramasına, türlerine göre filtrelemesine ve favorilerine eklemesine olanak tanır.


## Öne Çıkan Özellikler

*  **Dinamik Veri Akışı:** Gutendex API kullanılarak gerçek zamanlı kitap verileri çekilir.
*  **Akıllı Arama:** `useDebounce` ile optimize edilmiş, sunucu dostu arama deneyimi.
*  **Quick View (Hızlı Bakış):** Sayfadan ayrılmadan kitap detaylarını incelemeyi sağlayan animasyonlu modal yapısı.
*  **Responsive Tasarım:** Tailwind CSS ile mobil, tablet ve masaüstü cihazlara tam uyum.
* ❤️ **Favori Sistemi:** Beğenilen kitapların LocalStorage üzerinden yönetildiği kişisel liste.
* 🌗 **Dark/Light Mode:**  Göz yormayan tema desteği.



## Teknik Altyapı

* **Framework:** React 18 & Vite
* **Styling:** Tailwind CSS
* **Routing:** React Router v6
* **API/Veri:** Axios & Context API
